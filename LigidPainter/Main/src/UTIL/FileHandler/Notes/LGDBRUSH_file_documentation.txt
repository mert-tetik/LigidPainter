LigidPainter's brush file :

The Header

-3x64 bits (unsigned integer) description header -> 32 bits (unsigned integer) version value

--The description header : 0x8A3F9B0E5CFA2D71 , 0xE6724F8D319BCA06 , 0x597AEC82D60491B9 in the every version. 
--lgdbrush version value : 0x000007D0 (2000) (first version)

Brush Data

-32 bits (unsigned integer) size of the brush properties -> size of the properties times(
                                                                                            32 bits (unsigned integer) size of the property title ->
                                                                                            (8 bits * size of the property title) property title ->
                                                                                            8 bits property type ->
                                                                                            
                                                                                            if(property type == 'b')
                                                                                                1 bit property value
                                                                                            else if(property type == 'f')
                                                                                                32 bit (single-precision IEEE 754 number) property value
                                                                                            else if(value type == 't')
                                                                                                32 bits (signed integer) procedural ID -> 
                                                                                                32 bits (signed integer) is procedural inverted ->
                                                                                                32 bits (single-precision IEEE 754 number) procedural scale ->
                                                                                                32 bits (signed integer) texture width ->
                                                                                                32 bits (signed integer) texture height ->
                                                                                                texture width * texture height * 4 * sizeof(byte) bits (unsigned byte) texture data -> 
                                                                                        )
