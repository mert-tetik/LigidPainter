LigidPainter's material file :

The Header

-3x64 bits (unsigned integer) description header -> 32 bits (unsigned integer) version value -> 32 bits unqiue material ID (signed integer)

--The description header : 0x5A7D0C809A22F3B7 , 0xE8F691BE0D45C23A , 0x3FD8A9657B101E8C in the every version. 
--lgdmaterial version value : 0x000007D0 (2000) (first version)
--Material ID is not important 

Modifier Data

-64 bits (unsigned integer) size of the modifiers -> size of the modifiers times( 
                                                                                    (--Mask of the modifier--)
                                                                                    32 bits (signed integer) modifier type index -> 
                                                                                    32 bits (signed integer) procedural ID -> 
                                                                                    32 bits (signed integer) is procedural inverted ->
                                                                                    32 bits (single-precision IEEE 754 number) procedural scale ->
                                                                                    
                                                                                    32 bits (signed integer) texture width ->
                                                                                    32 bits (signed integer) texture height ->
                                                                                    texture width * texture height * 4 * sizeof(byte) bits (unsigned byte) texture data -> 
                                                                                    
                                                                                    (--Modifier data--)

                                                                                    32 bits (unsigned integer) property size->
                                                                                    
                                                                                    property size times(
                                                                                                            32 bits (unsigned integer) size of the title ->
                                                                                                            size of the title * sizeof(byte) bits modifier property title ->
                                                                                                            8 bits (signed char) value type -> 
                                                                                                            
                                                                                                            if(value type == 'f')
                                                                                                                32 bits (single-precision IEEE 754 number) property value
                                                                                                            else if(value type == '3')
                                                                                                                3 times (32 bits (single-precision IEEE 754 number) property value)
                                                                                                            else if(value type == 't')
                                                                                                                32 bits (signed integer) texture width ->
                                                                                                                32 bits (signed integer) texture height ->
                                                                                                                texture width * texture height * 4 * sizeof(byte) bits (unsigned byte) texture data -> 
                                                                                                        )
                                                                                )
